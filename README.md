# PDF Merger & Splitter

A simple Python project that can:

- Merge multiple PDFs
- Extract pages from PDFs

## Technologies Used

- Python
- pypdf

## Installation

pip install -r requirements.txt

## Run

python main.py

## Features

1. Merge PDFs
2. Split PDF Pages
